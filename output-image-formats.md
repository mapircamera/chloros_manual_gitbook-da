# Output-billedformater

### TIFF (16-bit)

16-bit (pixel digital nummer 0 - 65535) TIFF (.tif) format output.

### TIFF (32-bit, procent %)

32-bit (pixel float 0,0 - 1,0) TIFF (.tif) format output. Pixelværdier repræsenterer reflektansprocenter, hvor 0,0 = 0 % og 1,0 = 100 %.

### **JPG (8-bit)**

8-bit (pixel digital number 0 -255) JPG (.jpg) format output.

### **PNG (8-bit)**

8-bit (pixel digital number 0 - 255) PNG (.png) format output. Tabsfri komprimering, der er velegnet til visning og visualisering på internettet.
